# UTS Project: DompetKu - Expense Tracker

Aplikasi pencatat keuangan sederhana yang dibangun menggunakan React Native untuk memenuhi tugas Ujian Tengah Semester (UTS).

## Fitur Utama
- **Header Saldo Dinamis**: Menghitung total sisa saldo secara otomatis (Pemasukan - Pengeluaran).
- **Form Input Interaktif**: Mendukung pemisahan jenis transaksi langsung melalui dua tombol aksi eksekusi.
- **Riwayat Transaksi (FlatList)**: Menampilkan riwayat secara efisien dengan penanda warna visual khusus (Hijau untuk masuk, Merah untuk keluar).

## Cara Menjalankan Proyek
1. Clone repositori ini.
2. Jalankan perintah `npm install` untuk memasang semua dependencies.
3. Jalankan aplikasi menggunakan perintah `npx expo start` atau `npm start`.
